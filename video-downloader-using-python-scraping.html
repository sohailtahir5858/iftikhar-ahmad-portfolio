Social Media Video Downloader Using Web Scraping

This project focused on developing a web scraping tool for various social media platforms like Twitter, Facebook, Instagram, and more, to generate downloadable links for videos, along with metadata such as title, size, and quality. The tool was integrated into both web and mobile applications for client use.

Private Client

  • Industry: Social Media Tools
  • Size: 500+
  • Website: confidential
The client required a tool to enable their users to download videos from various social media platforms with proper metadata. This solution was integrated into both their web and mobile applications, providing a seamless user experience.
  • Requirement: Scrape video data from various social media platforms.
  • Requirement: Provide multiple download qualities (e.g., 360p, 720p).
  • Requirement: Seamlessly integrate with the client's web and mobile applications.

Project Overview

The project was aimed at building a system that could scrape video data from social media platforms and provide downloadable links. This included details like video title, size, quality, and type. The tool was designed to cater to multiple platforms such as Twitter, Facebook, and Instagram, offering support for various video formats and qualities.

The Challenge

A key challenge was ensuring that the scraping methods were always up to date and working for each platform, as social media sites frequently change their layouts and APIs. Additionally, ensuring that the API endpoints were secure and only accessible by authorized users was critical to prevent misuse.

The Approach & Solution

The backend was built using Python, Selenium, and undetected Chrome with headless browsing. A cron job was set up to check which scraping methods were functional every three hours, switching to the next working method if needed. The system was deployed on AWS Lambda for serverless execution, triggered via API Gateway. Proper security measures were implemented to restrict unauthorized access to the API.
