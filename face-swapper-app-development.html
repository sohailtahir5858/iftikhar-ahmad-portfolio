Face Swapping App Development

This project focused on creating a mobile face-swapping application that allows users to swap their faces with celebrity templates stored in the app. The solution integrates AI-based face-swapping models with mobile and cloud technologies for seamless real-time interactions.

Project Overview

The face-swapping app is a mobile application where users can upload their images and swap faces with predefined celebrity templates. The backend leverages pre-trained deepfake models (SimSwap) enhanced with Super-Resolution GAN (SRGAN) for better image quality. We optimized the solution by deploying it on AWS EC2 instances, with template images stored on AWS S3 and MongoDB for managing user data.

The Challenge

Ensuring high-quality face swaps while keeping processing time within a few seconds was a major challenge. The initial pre-trained models provided suboptimal results, requiring additional post-processing techniques. Another challenge was optimizing server costs, as the backend services ran 24/7, leading to significant expenses during the app's early stages when there were few active users.

The Approach & Solution

We improved the quality of face swaps by combining SimSwap with SRGAN for better visual fidelity. To optimize the backend cost, we transitioned from a 24/7 EC2 instance setup to an AWS Lambda-based solution using Docker containers. This shift reduced operational costs, as the Lambda function only runs when triggered by user requests. The images were stored in an AWS EFS, ensuring that the Lambda function could handle large files. The Docker image for the face-swapping model was hosted on AWS ECR, and we set up API Gateway for mobile app interactions.
